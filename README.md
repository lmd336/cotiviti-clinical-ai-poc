# cotiviti-clinical-ai-poc
# AI-Assisted Clinical Decision Support and Claim Review

A proof-of-concept prototype for Cotiviti's intern assessment. This project demonstrates how pattern recognition and agentic AI-style workflows can support healthcare Treatment, Payment, and Operations (TPO).

## Project Overview

This prototype analyzes a clinical note and demonstrates four core functions:

1. Detects likely diagnoses
2. Suggests possible ICD-10 codes
3. Flags missing documentation
4. Generates a reviewer-friendly clinical summary

## Topic

Clinical Decision Making and Pattern Recognition in Health Care: Chain Reasoning, Agentic Generative AI, Classification, Prediction, Inference, Clustering, and Time-Series Anomaly Detection for Treatment, Payment, and Operations.

## Technology Stack

- Python
- Streamlit
- Pandas
- Rule-based pattern recognition
- Modular agent-style workflow
